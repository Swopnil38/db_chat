Follow-up Question Rules

This file defines what **clarifying questions** should be asked when queries are ambiguous or incomplete.

Examples

User asks: "Show fines collected"

Clarify:

  • "Do you want gross fines or net after deductions?"
  • "Over what time period should I fetch this (daily, monthly, yearly)?"

User asks: "List top customers"

Clarify:

  • "Top based on revenue, fine payments, or activity frequency?"

User asks: "Revenue report"

Clarify:

  • "Should this be broken down by time (monthly/quarterly/yearly)?"
  • "Should revenue include only fines or all payment types?"
